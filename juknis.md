## 1. PENGERTIAN APLIKASI

### ðŸ’– Apa itu Posting Cinta?

**Posting Cinta** adalah singkatan dari **Posyandu Monitoring Cegah Stunting Anak**. Ini adalah aplikasi berbasis web yang dapat digunakan di handphone (HP) maupun komputer untuk membantu kader posyandu dalam memantau pertumbuhan anak.

### ðŸ“± Keunggulan Aplikasi

-   **Bisa Dipakai Offline**: Aplikasi tetap bisa digunakan meskipun tidak ada sinyal internet. Data akan tersimpan dan otomatis terkirim saat ada koneksi.
-   **Mudah Digunakan**: Tampilan sederhana dengan tombol-tombol besar, cocok untuk pengguna yang tidak terbiasa dengan teknologi.
-   **Sesuai Standar WHO**: Perhitungan status gizi menggunakan standar dari Organisasi Kesehatan Dunia (WHO) yang akurat.
-   **Bisa Dipasang di HP**: Aplikasi bisa dipasang di HP seperti aplikasi biasa (tanpa perlu download dari Play Store).

## 5. CARA PEMAKAIAN APLIKASI

### 5.1 Cara Masuk (Login) ke Aplikasi

**Langkah-langkah**:

1. **Buka Browser** di HP atau komputer (Chrome, Safari, atau Firefox)

2. **Ketik Alamat Website**:

    ```
    https://postingcinta.muaraenim.go.id
    ```

    (atau alamat yang diberikan oleh admin)

3. **Halaman Login Muncul**:

    - Masukkan **Email** yang sudah didaftarkan
    - Masukkan **Password**
    - Klik tombol **"Masuk"**

4. **Berhasil Login**: Anda akan masuk ke halaman Dashboard

> ðŸ’¡ **Tips**: Centang "Ingat Saya" agar tidak perlu login ulang setiap kali buka aplikasi.

### 5.2 Cara Memasang Aplikasi di HP (PWA)

**Untuk HP Android (Chrome)**:

1. Buka aplikasi di browser Chrome
2. Tunggu muncul banner **"Pasang Posting Cinta"** di bawah layar
3. Klik **"Pasang"** atau **"Install"**
4. Aplikasi akan muncul di layar utama HP seperti aplikasi biasa

**Untuk HP iPhone (Safari)**:

1. Buka aplikasi di browser Safari
2. Klik ikon **"Share"** (kotak dengan panah ke atas)
3. Pilih **"Add to Home Screen"** (Tambahkan ke Layar Utama)
4. Klik **"Add"** (Tambahkan)

### 5.3 Cara Menambah Data Ibu Baru

1. Dari Dashboard, klik menu **"Ibu"** di sidebar atau bottom navigation
2. Klik tombol **"+ Tambah Ibu"**
3. Isi data:
    - **Nama Lengkap**: Nama lengkap ibu
    - **NIK**: Nomor Induk Kependudukan (16 digit)
    - **Tanggal Lahir**: Tanggal lahir ibu
    - **Alamat**: RT/RW/Desa/Kecamatan
    - **No. HP**: Nomor handphone (opsional)
4. Klik tombol **"Simpan"**

### 5.4 Cara Menambah Data Anak Baru

1. Dari Dashboard, klik menu **"Anak"**
2. Klik tombol **"+ Tambah Anak"**
3. Isi data:
    - **Nama Lengkap**: Nama lengkap anak
    - **NIK**: NIK anak (opsional untuk bayi)
    - **Jenis Kelamin**: Pilih Laki-laki atau Perempuan
    - **Tanggal Lahir**: Tanggal lahir anak
    - **Nama Ibu**: Pilih dari daftar ibu yang sudah terdaftar
    - **Posyandu**: Pilih posyandu tempat anak terdaftar
4. Klik tombol **"Simpan"**

### 5.5 Cara Input Data Pengukuran

**Ini adalah fitur utama yang paling sering digunakan saat posyandu!**

1. Dari Dashboard, klik menu **"Pengukuran"**
2. Klik tombol **"+ Tambah Pengukuran"**
3. Isi data:
    - **Pilih Anak**: Ketik nama anak, lalu pilih dari daftar
    - **Tanggal Pengukuran**: Otomatis hari ini (bisa diubah jika perlu)
    - **Berat Badan (kg)**: Contoh: 10.5
    - **Tinggi Badan (cm)**: Contoh: 75.5
    - **Lingkar Kepala (cm)**: Contoh: 45.0 (opsional)
    - **Catatan**: Catatan tambahan jika ada (opsional)
4. Klik tombol **"Simpan"**

**Setelah disimpan, sistem akan otomatis**:

-   Menghitung Z-Score (nilai status gizi)
-   Menentukan status gizi (Normal/Stunting/Wasting/dll)
-   Memperbarui grafik pertumbuhan anak

> âš ï¸ **Penting**: Pastikan berat badan dan tinggi badan diisi dengan benar! Kesalahan input akan mempengaruhi hasil perhitungan status gizi.

### 5.6 Cara Melihat Grafik Pertumbuhan Anak

1. Dari menu **"Anak"**, klik nama anak yang ingin dilihat
2. Di halaman detail anak, klik **"Lihat Grafik"**
3. Akan muncul 3 jenis grafik:
    - **Grafik BB/U**: Berat Badan menurut Umur (deteksi Underweight)
    - **Grafik TB/U**: Tinggi Badan menurut Umur (deteksi Stunting)
    - **Grafik BB/TB**: Berat Badan menurut Tinggi Badan (deteksi Wasting)

**Cara Membaca Grafik**:

-   **Garis Hijau**: Area normal
-   **Garis Kuning**: Area berisiko
-   **Garis Merah**: Area berbahaya
-   **Titik Biru**: Posisi anak Anda

### 5.7 Cara Melihat Resep Makanan Sehat

1. Klik menu **"Resep"** di sidebar atau bottom navigation
2. Pilih kategori:
    - **MPASI**: Untuk bayi 6-12 bulan
    - **Balita**: Untuk anak 1-3 tahun
    - **Anak**: Untuk anak 3-5 tahun
3. Klik resep yang ingin dilihat
4. Akan muncul:
    - Bahan-bahan
    - Cara membuat
    - Informasi gizi

### 5.8 Cara Menggunakan Aplikasi Saat Offline

**Aplikasi tetap bisa digunakan meski tidak ada internet!**

1. Saat tidak ada sinyal, akan muncul banner **"Mode Offline"** berwarna kuning
2. **Tetap bisa dilakukan**:
    - Input data pengukuran
    - Lihat data anak yang sudah di-cache
    - Lihat resep yang sudah pernah dibuka
3. Data akan **tersimpan sementara** di HP
4. Saat ada sinyal internet, data akan **otomatis terkirim** ke server

> ðŸ’¡ **Tips**: Sebelum ke posyandu, buka aplikasi saat ada WiFi agar data anak ter-cache.
